# Cataract Detection using Deep Learning

This project uses Convolutional Neural Networks (CNNs) to detect cataracts from retinal fundus images.

## Models Used

- VGG19
- ResNet50
- MobileNetV2

## Dataset

ODIR-5K (Ocular Disease Intelligent Recognition Dataset)

## Technologies

Python
TensorFlow
Keras
OpenCV
NumPy
Matplotlib

## Methodology

1. Image preprocessing
2. Data augmentation
3. Transfer learning
4. CNN training
5. Model evaluation

## Results

ResNet50 achieved the highest accuracy.
